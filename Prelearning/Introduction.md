## üõ†Ô∏è Debugging Checklist

When you encounter problems, work through this checklist systematically:

### 1. **Read the Error Message**
- [ ] What does the error message say?
- [ ] Which line is causing the problem?
- [ ] What type of error is it? (SyntaxError, TypeError, ValueError, etc.)

### 2. **Check Your Code**
- [ ] Are all parentheses, brackets, and quotes properly closed?
- [ ] Are you using the correct variable names?
- [ ] Are you using the right data types?
- [ ] Is your indentation correct?

### 3. **Test Small Parts**
- [ ] Can you run a simpler version of your code?
- [ ] Does each part work individually?
- [ ] Can you print out intermediate values to see what's happening?

### 4. **Research the Problem**
- [ ] Have you searched for the exact error message?
- [ ] Have you looked up the functions you're using?
- [ ] Have you asked Cursor for help with the specific error?

### 5. **Try Different Approaches**
- [ ] Can you solve the problem a different way?
- [ ] Are there alternative functions or methods you could use?
- [ ] Can you break the problem into smaller steps?
